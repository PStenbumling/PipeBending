# v0.5.5 Rules — Human Error Prevention

RULE: Prevent errors before calculation, not after.
RULE: Do not rely on user attention.
RULE: All warnings must be consequence-first.
RULE: Clamp violation is physically impossible and must be red.
RULE: Lead-in clamp warning overrides tail clamp warning.
RULE: Never show both lead-in and tail clamp warnings simultaneously.
RULE: Reference direction must always be visible.
RULE: Reference must exist in both input and result zones.
RULE: Bananböj reference origin is H-stycke end.
RULE: Tool mismatch warning must persist until corrected.
RULE: Tool mismatch warning must show real-world consequence.
RULE: Estimated values must be visibly marked as estimates.
RULE: Estimated value counter is global within the session.
RULE: Estimated badge must escalate after three uses.
RULE: Red warnings suppress all amber warnings.
RULE: Red warnings must visually dominate.
RULE: Kaplängd equals last mark position rounded.
RULE: No independent kaplängd calculation is allowed.
RULE: Marks are absolute positions, not segment lengths.
RULE: Offset must be measured perpendicular centerline-to-centerline.
RULE: If D >= L × 0.95, show likely measurement error warning.
RULE: If angle is below 20 degrees, show shallow-angle warning.
RULE: Manual override must be validated.
RULE: If override is greater than pipeSize × 3, show warning.
RULE: If override approximates CLR, show warning.
RULE: Blocking UI is not allowed.
RULE: Warnings persist until condition is resolved.
RULE: Instick visuals must not override reference origin.
RULE: Reference marker must be visually dominant.
RULE: Do not introduce new math.
RULE: Do not change geometry engine.
