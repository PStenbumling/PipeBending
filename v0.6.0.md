## Measurement Model (Real-world)

START:
Mått tas från vägg till rörplan.
Plane offset och instick måste därför adderas innan böjavdrag dras av.

```
adj[0] = +planeOffset + insertionDepth - bendDeduction
```

MIDDLE:
Rörplan till rörplan.

```
adj[i] = -bendDeduction
```

END (CC):
Mått tas från rörplan till kopplingens centrum.

```
adj[last] = -planeOffset - La_slut
```

Regler:
- START använder `planeOffset` + `insertionDepth`
- MIDDLE använder bara böjavdrag
- END använder `planeOffset` + `La`
- `planeOffset` används endast på START och END
- Ingen mellansträcka påverkas av planbyte

## Justeringslogik

**Start:**
```
adj[0] = +planeOffset + insertionDepth - bendDeduction
```

**Mellan:**
```
adj[i] = -bendDeduction
```

**Slut — ej CC:**
```
adj[n−1] = -planeOffset + La_slut
```

**Slut — CC:**
```
adj[n−1] = -planeOffset - La_slut
```

Plane offset appliceras aldrig på mellansträckor.

## UI (inline i sträckraden)

```
Sträcka 1  [ 232 ]  [VA 1130 Skarv ▼]  [ ← | → ]  [×]
Sträcka 2  [ 405 ]                     [ ← | → ]  [×]
Sträcka 3  [ 133 ]  [VA 1150 T-stycke ▼] [☑ CC]   [×]
```

- Kopplingsval visas på start-rad (rad 1) och slut-rad (sista raden)
- Mellansträckor har ingen kopplingskolumn
- Väljer användaren koppling → instick/La uppdateras automatiskt från fittings.json
- Väljer användaren "— ingen —" → La = 0, ingen justering
- `defaultCC` i fittings.json sätter CC-checkboxen automatiskt
- CC-checkbox visas alltid på slutraden, oavsett läge
- La-värde hanteras internt (ej synligt fält)
- Offset-fältet i UI motsvarar `planeOffset` (vägg → rörplan)

## Dataflöde

```
fittings.json  →  mbFittings[]  (fetch, window.onload)
      │
mbFittingSelected('start')
      └─ insertionDepth  ← insertionDepth[Dy]
      │
mbFittingSelected('slut')
      ├─ La  ← dimensioner[Dy].La
      └─ CC  ← defaultCC
      │
mbRecomputeAdjustments()
      └─ start: +planeOffset + insertionDepth − bend deduction
      └─ mellan: − bend deduction
      └─ slut CC: −planeOffset − La
```

## Referensfall

**Scenario:** Skarvkoppling (start) → T-stycke CC (slut)

| Sträcka | Bas | Typ | Justering | Effektiv |
|---------|-----|-----|-----------|---------|
| S1 | 232 | start, vägg → rörplan | +offset + instick − böjavdrag | 232 + 20 + 10 − 19 = 243 |
| S2 | 405 | mellansträcka | − böjavdrag | 405 − 19 = 386 |
| S3 | 133 | slut, rörplan → CC | −offset − La_slut | 133 − 20 − 12 = 101 |

planeOffset = 20 mm, bend deduction = 19 mm, insertionDepth_start = 10 mm (VA 1130 / 12mm), La_slut = 12 mm (VA 1150 / 12mm)

## Bakåtkompatibilitet

Default (ingen koppling vald, CC = av) använder fortfarande samma kopplingsmodell men med `planeOffset` bara på START och SLUT.
