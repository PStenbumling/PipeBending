# v0.6.0 — Kopplingsdata + La-autofyll + CC-hantering

---

## Skillnad mot v0.5.8

| | v0.5.8 | v0.6.0 |
|---|---|---|
| La-källa | Globalt fält | Dropdown → fittings.json per Dy |
| CC-slutände | Ej hanterat | `adj[n-1] = −La_slut` (ersätter) |
| Justeringsformel start | `−build + La` | Oförändrad |
| Justeringsformel slut (ej CC) | `+build + La` | Oförändrad |
| Mellansträckor | Oförändrad | Oförändrad |

---

## Justeringslogik

**Start (oförändrad):**
```
adj[0] = −build + La_start
```

**Slut — ej CC:**
```
adj[n−1] = +build + La_slut
```

**Slut — CC (ersätter hela formeln):**
```
adj[n−1] = −La_slut
```

Build appliceras aldrig på slutände när CC är aktivt.

## UI (Avancerat läge)

```
START-KOPPLING                    SLUT-KOPPLING
[— ingen — ▼]  La: 10 mm         [VA 1150 T-stycke ▼]  La: 12 mm
                                  [✓] Mätt till CC
```

- Väljer användaren koppling → La fylls från fittings.json, fältet låses
- Väljer användaren "— ingen —" → La-fältet fritt redigerbart
- `defaultCC` i fittings.json sätter CC-checkboxen automatiskt
- CC-checkboxen kan ändras manuellt efteråt
- Hela sektionen dold i Enkelt läge

---

## Dataflöde

```
fittings.json  →  mbFittings[]  (fetch, window.onload)
      │
mbFittingSelected('slut')
      ├─ La  ← dimensioner[Dy].La
      └─ CC  ← defaultCC
      │
mbRecomputeAdjustments()
      └─ isCC ? adj[n-1] = −La : adj[n-1] = +build + La
```

## Referensfall

**Scenario:** Skarvkoppling (start) → T-stycke CC (slut)

| Sträcka | Bas | Typ | Justering | Effektiv |
|---------|-----|-----|-----------|---------|
| S1 | 232 | start, skarvkoppling | −build + La_start | 232 − 20 + 1 = 213 |
| S2 | 405 | mellansträcka | −build | 405 − 20 = 385 |
| S3 | 133 | slut, T-stycke CC | −La_slut | 133 − 12 = 121 |

build = 20 mm, La_start = 1 mm (VA 1130 / 12mm), La_slut = 12 mm (VA 1150 / 12mm)
